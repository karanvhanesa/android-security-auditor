Basics of Cyber Security & Linux

What is Cyber Security

Cyber Security is the practice of protecting computers, networks, systems, and data from digital attacks, unauthorized access, or damage. These attacks are usually aimed at accessing, changing, or destroying sensitive information, extorting money, or interrupting normal business processes.

In today’s digital world, almost everything depends on the internet such as banking, communication, social media, and business operations. Because of this, cyber threats are increasing every day. Cyber Security helps individuals and organizations protect their digital assets from hackers and malicious activities.

Cyber security includes multiple domains such as network security, application security, information security, cloud security, mobile security, and ethical hacking. The main goal is to maintain the CIA triad:

  • Confidentiality: Protecting sensitive information from unauthorized access.
  • Integrity: Ensuring data is not modified or tampered with.
  • Availability: Ensuring systems and data are accessible when needed.

Cyber security is important because it supports trust in digital systems and reduces the risk of theft, disruption, and privacy loss.

What is a Hacker

A hacker is a person who uses technical skills to understand computer systems, networks, and applications. Hackers often try to find vulnerabilities or weaknesses in systems. The term hacker does not always mean a criminal. Some hackers use their skills to improve security and protect systems.

Hackers may work for cyber security companies, government agencies, bug bounty programs, ethical hacking firms, and security research teams. They usually have strong knowledge of operating systems, networking, programming, security vulnerabilities, and web technologies.

Types of Hackers

White Hat Hacker: White hat hackers are ethical security professionals who use hacking techniques legally to identify vulnerabilities in systems. Their goal is to improve security and protect organizations from cyber attacks.

Black Hat Hacker: Black hat hackers perform illegal hacking activities. Their goal is usually to steal data, gain unauthorized access, or cause damage through activities such as ransomware, fraud, and website defacement.

Grey Hat Hacker: Grey hat hackers fall between white hat and black hat hackers. They may access systems without permission but usually do not have malicious intentions and may later report vulnerabilities.

What is an Operating System (OS)

An Operating System is system software that manages computer hardware and software resources. It acts as an interface between the user and the computer hardware. Without an operating system, a computer cannot function properly for regular use.

The operating system performs several tasks such as managing hardware resources, running applications, managing files and storage, handling input and output devices, and managing memory and processes.

Examples of operating systems include Windows, Linux, macOS, Android, and iOS. Operating systems can be categorized based on their usage:

  • Desktop Operating System: Used for personal computers and laptops.
  • Mobile Operating System: Designed for smartphones and tablets.
  • Server Operating System: Used to manage servers and network resources.
  • Embedded Operating System: Used in routers, smart TVs, and IoT devices.

What is Linux

Linux is an open-source operating system based on Unix architecture. It is widely used by developers, system administrators, and cyber security professionals. Linux is free, customizable, and highly secure. Because of these features, it is widely used in cyber security and penetration testing.

Linux allows users to modify the source code and customize the system according to their needs. Its flexibility and command-line power make it suitable for both general computing and advanced technical work.

Why Linux is Used in Hacking

  • Open Source: The source code is publicly available and can be modified.
  • Powerful Command Line: Linux provides an efficient terminal for complex tasks.
  • Security and Stability: Linux systems are generally secure and reliable.
  • Tool Availability: Many cyber security tools are built for Linux environments.
  • Customization: Linux allows complete control over system configuration.

Linux Distributions (Distros)

A Linux distribution is a version of Linux that includes the Linux kernel along with additional software and tools. Different distributions are designed for different purposes such as development, penetration testing, or general use.

Examples of Linux distributions include Kali Linux, Ubuntu, Parrot OS, Debian, Fedora, and Arch Linux. Each distribution serves a specific purpose and gives users a different balance of simplicity, customization, and tools.

Kali Linux

Kali Linux is one of the most popular Linux distributions used for cyber security and penetration testing. It is developed and maintained by Offensive Security. Kali Linux contains hundreds of pre-installed security tools used for penetration testing, digital forensics, vulnerability assessment, wireless attacks, and security research.

Popular tools in Kali Linux include Nmap, Burp Suite, Metasploit, Wireshark, John the Ripper, and Aircrack-ng.

Lab Setup for Kali Linux

Before practicing hacking techniques, it is important to create a safe and controlled lab environment. A cyber security lab allows students to practice attacks and defenses without harming real systems. Common lab components include Kali Linux as the attacker machine, a vulnerable machine as the target, and virtualization software such as VirtualBox or VMware.

Virtual lab environments allow safe experimentation and easy recovery through snapshots. A beginner should always work inside a controlled environment and never test against real systems without permission.

Safe practice matters. Students should test only in their own labs or with explicit written authorization, keep systems isolated from production networks, and document their setup carefully.

Types of Kali Linux Installation

Virtual Machine Installation

Safe testing environment, easy to install, and no need to modify the main operating system.

Dual Boot Installation

Lets users install Kali Linux alongside another operating system and choose one during boot.

Live Boot (USB)

Portable and does not permanently change the host system.

Bare Metal Installation

Direct installation on the machine for full hardware access and maximum performance.

Ethical Hacking and Learning Path

To become a cyber security professional, students should focus on learning networking fundamentals, the Linux operating system, web technologies such as HTTP, HTML, and JavaScript, vulnerability assessment, penetration testing, and security tools. Practical learning through labs, Capture the Flag challenges, and responsible real-world testing is essential.

A strong learning path begins with understanding how systems and networks work, then moves into Linux administration, web basics, reconnaissance, vulnerability discovery, and controlled exploitation. Ethical conduct is a core requirement at every step.
